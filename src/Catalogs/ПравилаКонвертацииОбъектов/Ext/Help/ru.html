Справочник «Правила конвертации объектов»

Справочник «Правила конвертации объектов» подчинен справочнику «Конвертации». Каждый элемент справочника описывет правило формирования образа объекта, определенного типа, конфигурации приемника в файле. В качестве источника данных может выступать объект определенного типа конфигурации источника или произвольные данные. Таким образом, основными реквизитами правила конвертации объектов являются реквизиты Объект-источник и Объект-приемник, ссылающиеся на элементы справочника "Объекты метаданных" конфигураций источника и приемника соответственно. Для удобства правила конвертации можно объединять в группы. Если в настройках правила установить флажок "Не замещать найденные объекты при загрузке", то объекты идентифицированные при загрузке (т.е. найденные в информационной базе приемнике по значению синхронизирующих полей) не будут изменяться. Универсальная обработка выгрузки данных хранит все выгруженные объекты в специальном списке и, таким образом, многократно встречающиеся объекты (по ссылкам из реквизитов других объектов) выгружаются единожды. Однако, установив флажок "Не запоминать выгруженные объекты" в настройках правила, можно отключить этот режим. Данный режим может быть полезен для оптимизации, например, когда точно известно, что на выгружаемые объекты нет ссылок, а так же в случае, когда на основании одного объекта-источника в зависимости от различных условий могут формироваться несколько различных объектов-приемников.

Табличная часть "Варианты настроек полей поиска" не используется для механизма обмена данными, реализованного в конфигурации 1С:Библиотека стандартных подсистем. Вместо нее необходимо использовать помощник интерактивного обмена данными.

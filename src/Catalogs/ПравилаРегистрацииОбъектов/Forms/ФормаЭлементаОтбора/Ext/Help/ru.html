Форма служит для определения значения константы.
Значение константы можно определить двумя способами: в поле формы для примитивных типов данных* или написать алгоритм получения значения. Результатом вычисления алгоритма является определение параметра Значение.
* в качестве примитивных типов используются следующие типы: Дата, Число, Строка, Булево.

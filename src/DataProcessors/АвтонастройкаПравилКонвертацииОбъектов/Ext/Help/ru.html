Обработка "Автонастройка правил конвертации объектов".

Обработка предназначена для автонастройки правил конвертации объектов.

Объекты в дереве сгруппированы по типу. В колонке "Тип/Правило" на первом уровне отображается тип объектов, а на втором уровне существующие правила конвертации объектов. В колонке "Объект источник" отображается объект конфигурации источника, а в колонке "Объект приемник" отображается объект конфигурации приемника. Если объект источник и приемник синхронизированы, то в колонке "Имя нового правила" отображается имя, которое будет присвоено новому правилу (возможно его изменение) и в колонке "Тип/Правило" установлен флажок создания нового правила конвертации объектов. Для несинхронизированных автоматически объектов источника или приемника может быть установлено соответствие вручную. Указание объекта приемника является обязательным для возможности создания правила конвертации объектов. Редактирование существующих правил в данном режиме невозможно. По окончании автоматической и, по необходимости, ручной синхронизации возможно установить флажки создания нового правила конвертации объектов только для необходимых объектов. При записи правила конвертации объектов проверяется уникальность имени в пределах настраиваемой конвертации и в случае обнаружения дублей имен запись не производится. В данном случае необходимо вручную изменить имя, задав свое уникальное имя в колонке "Имя нового правила".

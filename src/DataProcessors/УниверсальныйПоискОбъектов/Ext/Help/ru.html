Обработка "Универсальный поиск объектов"

Обработка "Универсальный поиск объектов" предназначена для поиска элементов справочников.

Пользователь имеет возможность указать тип объектов для поиска, строку поиска и области данных где нужно искать. Для каждой области данных можно указать поля в которых необходимо производить поиск строки, а так же дополнительные ограничения.

Имеется возможность организовать поиск объектов, входящих хотя бы в одну из областей поиска, или поиск объектов удовлетворяющих уловиям поиска для всех областей. В качестве результата поиска можно получить либо список найденных объектов, либо детализированное дерево поиска, в котором для каждого найденного объекта указана область и поля, где он был найден. Такую информацию можно получить для каждого найденного объекта и при нажатии на кнопку "Детальный поиск".

Обработка может быть настроена на поиск объекта произвольного справочника. Первоначальные настройки поиска хранятся в макете обработки "ФорматПоиска".

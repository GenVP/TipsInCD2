Обработка "Автонастройка правил конвертации свойств".

Обработка предназначена для автонастройки правил конвертации свойств.

Свойства в дереве расположены в таком же иерархическом порядке как и в структуре метаданных конфигурации. Группами описываются подчиненные объекты: табличные части и движения документов. В колонке "Правило КС" отображаются существующие правила конвертации свойств. В колонке "Источник" отображается свойство объекта конфигурации источника, а в колонке "Приемник" отображается свойство объекта конфигурации приемника. Если свойство объекта источника и приемника синхронизированы, то в колонке "Правило КС" установлен флажок создания нового правила конвертации свойств. Для несинхронизированных автоматически свойств объектов источника или приемника может быть установлено соответствие вручную. При выборе группы источника/приемника (например, табличных частей), автоматически производится синхронизация подчиненных объектов (например, реквизитов табличных частей). Указание свойства объекта приемника является обязательным для возможности создания правила конвертации свойств. Редактирование существующих правил в данном режиме невозможно. Для синхронизированных свойств объектов (только в случае несоставного типа данных источника и приемника) автоматически осуществляется поиск правила конвертации объектов и найденное правила конвертации объектов отображается в колонке "Правило КО". По окончании автоматической и, по необходимости, ручной синхронизации возможно установить флажки создания нового правила конвертации свойств только для необходимых свойств объектов. Для того чтобы при создании правила конвертации свойств создавались правила конвертации объектов (только в случае несоставного типа данных источника и приемника) необходимо установить флажок "Создавать правила КО".

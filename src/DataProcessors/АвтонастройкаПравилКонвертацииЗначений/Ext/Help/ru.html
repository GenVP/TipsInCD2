Обработка "Автонастройка правил конвертации свойств".

Обработка предназначена для автонастройки правил конвертации свойств.

В колонке "Правило КЗ" отображаются существующие правила конвертации значений. В колонке "Источник" отображается значение объекта конфигурации источника, а в колонке "Приемник" отображается значение объекта конфигурации приемника. Если значения объекта источника и приемника синхронизированы, тогда в колонке "Правило КЗ" установлен флажок создания нового правила конвертации значений. Для несинхронизированных автоматически значений объектов источника или приемника может быть установлено соответствие вручную. Указание значения объекта приемника является обязательным для возможности создания правила конвертации значений. Редактирование существующих правил в данном режиме невозможно. По окончании автоматической и, по необходимости, ручной синхронизации возможно установить флажки создания нового правила конвертации значений только для необходимых значений объектов.

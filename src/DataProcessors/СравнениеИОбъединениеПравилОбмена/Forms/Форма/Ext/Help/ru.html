Обработка предназначена для выполнения сравнения правил обмена между собой и объединения их как в целом друг с другом, так и их отдельных элементов (справочников, составляющих правила обмена).

Выполнение сравнения правил.


Для того чтобы выполнить сравнение правил необходимо открыть форму конвертации и выполнить команду "Сервис - Сравнение и объединение правил обмена". После выполнения команды открывается форма выбора конвертации для сравнения. Выбранная конвертация сравнивается с текущей. Процесс сравнения отображается с помощью формы индикации. Результат сравнения будет выведен в виде дерева значений в табличное поле формы. Порядок следования колонок в табличном поле дерева сравнения зависит от порядка следования строк в табличной части "Конвертации", которая настаивается в форме настройки. При изменении состава строк этой табличной части дерево сравнения будет очищено и анализ будет необходимо выполнить заново. Для объединения правил обмена предназначена команда "Выполнить".
Команда"Использовать подвал для анализа различий" позволит использовать подвал табличного поля для просмотра содержимого текущей строки дерева сравнения. При ее нажатии появится полоса регулирования, с помощью которой можно управлять высотой подвала.
Отображение дерева сравнения в табличном поле может быть ограничено значением фильтра отбора. При изменении фильтра повторный анализ различий между текущими правилами обмена не производится. Он может принимать следующие значения:

  • Показывать все. Будут выведены все строки дерева сравнения. Строки в которых найдены отличия выводятся жирным шрифтом.
  • Показывать измененные. Выводятся все строки, в которых обнаружены изменения, в том числе добавленные и удаленные.
  • Показывать добавленные и удаленные. Выводятся только строки, которые были добавлены или удалены.

Работа с деревом сравнения.


После выполнения сравнения в табличном поле будут доступны следующие предопределенные узлы:

  • Общие. Содержит ссылку на правило обмена. Подчиненные строки включают реквизиты справочника "Конвертации", узлы и реквизиты справочника "Параметры".
  • Правила конвертации объектов. В подчиненных строках находятся ссылки на элементы справочников "Правила конвертации объектов", "Правила конвертации свойств" и "Правила конвертации значений, а также их реквизиты.
  • Правила выгрузки данных.
  • Правила очистки данных.
  • Алгоритмы.
  • Запросы.

Узел может быть не представлен в табличном поле, если он не содержит строк, удовлетворяющих значению текущего фильтра отбора.

Состав команд контекстного меню в зависимости от текущей колонки может содержать следующие команды:

  • Свернуть. При наличии подчиненных строк.
  • Развернуть. При наличии подчиненных строк.
  • Выполнить поиск различий. Выполняется анализ различий текущей строки и всех подчиненных.
  • Выполнить объединение. Производится частичное объединение правил обмена. После выполнения объединения производится анализ различий в текущей строке и подчиненных. Пометки с них снимаются.
  • Перейти к правилам. Выполняется открытие формы настройки правил обмена и установка курсора на строку текущего элемента.
  • Сравнить модули выгрузки (загрузки). Доступен только в корневом узле дерева анализа для правил, источником или приемником которых является конфигурация на платформе 1С:Предприятие 7.7

Настройка сравнения и объединения правил обмена.


Настройка обработки выполняется в специальной форме настройки, перейти к которой можно при нажатии кнопки "Настройка". Имеется возможность настроить правила сравнения объектов, составляющих правила обмена - выбрать справочники и их реквизиты, для которых будет выполняться сравнение, выбрать правила обмена, которые будут сравниваться между собой. Для того чтобы выбранные настройки вступили в силу необходимо выполнить команду "Обновить".

Форма предназначена для выполнения настройки обработки "Сравнение и объединение правил обмена" и состоит из двух частей:

  • Выполнение общей настройки. Вызывается при открытии формы по кнопке "Настройка" основной формы обработки.
  • Выполнение настройки правил обмена. Открывается при нажатии на гиперссылку "Перейти к настройке правил обмена".

Общая настройка сравнения и объединения.

На первом этапе необходимо выбрать разделы учета, для которых будет выполняться сравнение.

Для выбора доступны следующие значения:

  1. Общие настройки правил обмена. Включает сравнение реквизитов, обработчиков событий и параметров элементов справочника "Конвертации".
  2. Правила конвертации объектов. Включает сравнение элементов справочника "Правила конвертации объектов".
    1. Правила конвертации свойств. Включает сравнение элементов справочника "Правила конвертации свойств". Для выбора доступны следующие режимы сопоставления элементов справочника - "по коду" и "по наименованию".
    2. Правила конвертации значений. Включает сравнение элементов справочника "Правила конвертации значений".
  3. Правила выгрузки данных. Включает сравнение элементов справочника "Правила выгрузки данных".
  4. Правила очистки данных. Включает сравнение элементов справочника "Правила очистки данных".
  5. Алгоритмы. Включает сравнение элементов справочника "Алгоритмы".
  6. Запросы. Включает сравнение элементов справочника "Запросы".

При снятии / установке флажка напротив соответствующего раздела изменяется состав доступных разделов настройки сравнения.

При установке флажка "Сравнивать реквизиты ссылочного типа по их представлению" сравнение реквизитов ссылочного типа будет производиться по основному представлению. В противном случае, будет сформирована строка, содержащая представления всех реквизитов этого объекта, и только после этого будет выполнено сравнение ее с аналогичным образом подготовленным значением другого реквизита сравнения.

После выбора разделов сравнения есть возможность настроить реквизиты этих справочников, которые будут участвовать в сравнении. Для этого необходимо щелкнуть мышкой по сответствующему разделу в поле списка разделов и перейти на страницу детальной настройки реквизитов сравнения.

Для того, чтобы перед объединением правил обмена обработка запрашивала подтверждение - установите флажок "Требовать подтверждения перед объединением".

При выполнении объединения объектов средствами 1С:Предприятие используется полное замещение свойств объекта - приемника свойствами объекта - источника. Для алгоритмов, запросов и обработчиков имеется возможность использовать программу KDiff3 для выполнения более гибкого объединения строк. Для использования KDiff3 установите соответствующий флажок и укажите путь к исполняемому файлу программы. 

Настройка правил обмена.

На этом этапе необходимо выбрать правила обмена, которые будут участвовать в сравнении. Для заполнения списка правил можно воспользоваться кнопкой "Подбор". Порядок следования строк правил обмена будет определять порядок следования колонок в дереве сравнения. Изменение порядка следования строк приведет к очистке строк дерева сравнения.

При выборе правила обмена сохраняются последние 10 значений, которые будут доступны при выборе из списка.

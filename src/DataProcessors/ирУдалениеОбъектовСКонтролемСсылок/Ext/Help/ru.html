Обработка "Удаление помеченных объектов"

Обработка "Удаление помеченных объектов" предназначена для удаления объектов, помеченных на удаление.

Данная обработка является аналогом стандартной обработки "Удаление помеченных объектов". Позволяет удалять элементы в раздельном режиме работы "1С:Предприятие 8.0". Имеет раширенный отбор по объектам метаданных.

ВНИМАНИЕ!!!
При использовании обработки в раздельном режиме работы, может нарушиться логическая целостность информационной базы. Чтобы исключить возникновение этой ошибки, предупредите всех активных пользователей не записывать объекты, имеющие в своем составе реквизитов - помеченные на удаление. 

  

В верхней части диалог содержит список обнаруженных в информационной базе объектов, помеченных на удаление. Любой из объектов можно открыть с помощью кнопки Изменить.

С помощью пометок можно указать, какие объекты нужно проверить на возможность удаления.

При помощи отбора можно отобрать только определенные объекты метаданных. Для этого нужно воспользоваться кнопкой Отобрать по категориям. В открывшейся форме на дереве значений установить флажки только интересующим объектам метаданных.

Для контроля наличия ссылок на удаляемые объекты нажмите кнопку Контроль.
Программа формирует список ссылок на указанные объекты и выводит их в  нижнем списке по каждому выбранному объекту, помеченному на удаление. Любую из ссылок можно открыть с помощью кнопки Изменить.

Во время просмотра списка помеченных на удаление объектов в информационной базе могут произойти изменения. Чтобы обновить список нажмите кнопку Обновить.

Для удаление объектов, которые возможно удалить используйте кнопку Удалить.

Для выполнения обработки в пакетном режиме после нажатия на  кнопку Контроль на вопрос "Выполнить после "Контроля"   "Удаление" объектов, которые возможно удалить?" ответьте ДА.

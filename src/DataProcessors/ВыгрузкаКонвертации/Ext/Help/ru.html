Обработка "Выгрузка конвертации".

Обработка предназначена для сохранения правил конвертации в XML файл.

В диалоге обработки необходимо указать имя файла выгрузки правил обмена, выбрать конвертацию и выполнить сохранение.

Если в качестве источника выступает конфигурация "1С:Предприятие версии 7.7", то необходимо так же выгружать в текстовый файл модуль (флаг "Выгружать модуль"), предварительно указам путь к файлу в реквизите "Имя файла модуля". Перед выгрузкой данных из "1С:Предприятия версии 7.7" необходимо полностью заменить модуль обработки "Универсальная выгрузка данных в формате XML (V77Exp.ert)" полученным текстом.

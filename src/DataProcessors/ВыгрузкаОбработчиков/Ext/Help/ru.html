Позволяет использовать все возможности, предоставляемые конфигуратором, для отладки и написания кода обработчиков. Формируемый модуль предназначен для отладки и содержит в себе все служебные процедуры и функции, в том числе и неиспользуемые. Для формирования кода модуля, не содержащего неиспользуемых процедур и функций воспользуйтесь командой сохранения правил конвертации.

Для формирования отладочного модуля необходимо:

1. Скопировать сформированный текст модуля в буфер обмена.

2. Открыть в указанной базе внешнюю обработку или создать новую.

3. Вставить в модуль обработки текст из буфера обмена.

4. Сохранить эту обработку. При сохранении не должно выдаваться ошибок компиляции.

Примечание: Для того чтобы воспользоваться возможностями режима отладки необходимо явно задать параметры алгоритмов.

5. Подключить эту обработку к конфигурации на базе БСП не ниже версии 2.1.

Отчет "Отчет по правилам обмена данными".

Отчет предназначен для получения подробной информации по правилам обмена данными для выбранной конфигурации.

Данный отчет предназначен как для разработчиков, так и для пользователей конфигурации. Также, он служит инструментом проверки разработанных правил на корректность составления, наличие описаний нетиповых правил.

Форма настроек отчета "Отчет по правилам обмена данными"

 

В данной форме расположены все возможные варианты настройки данного отчета.

Сначала следует выбрать с помощью настроек необходимый набор данных. В варианте по умолчанию (когда все флаги сброшены) будет выведена информация лишь об объектах, которые фигурируют в источниках и/или приемниках правил обмена. Дополнительные данные можно получить, установив один или несколько флагов в группах "Выводить для источника" и "Выводить для приемника"

Опция "Для составных описаний типов выводить не более <n> описаний типов" предназначена для удобства пользовательского представления отчета. В случае, если какой-то объект имеет составной тип данных, состоящий из большого количества типов, его представление в отчете будет громоздким. При включении данной опции программа сократит описание его типа до указанного количества типов.

Опция "Выводить сообщения об ошибках правил" представляет ценность для разработчиков правил обмена данных. При включении данной области программа выведет в отчет предупреждения о некорректно заполненных или настроенных правилах, входящих в состав данной конвертации. При этом, из детального описания правил будет доступен переход к самому элементу правил по гиперссылке для оперативного его исправления..
